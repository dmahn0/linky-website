# Linky Platform

## 🎨 UI 개발 = ui-core.html 복사/붙여넣기

### 핵심 파일
- **`ui-core.html`** - 34개 컴포넌트 모두 완성 (여기서 복사)
- **`docs/DESIGN_SYSTEM.md`** - 디자인 규칙
- **`CLAUDE.md`** - 개발 가이드

### 사용법
1. ui-core.html 열기
2. 원하는 컴포넌트 찾기 (Ctrl+F)
3. HTML 복사
4. 붙여넣기
5. 끝!

### 프로젝트 구조
```
linky-platform/
├── ui-core.html          # ⭐ 모든 UI 여기 있음
├── CLAUDE.md             # 개발 가이드
├── config/               # 설정 파일들
└── docs/                 # 문서
    └── DESIGN_SYSTEM.md  # 디자인 규칙
```
